Quick Create Portal User Access for Contacts

Create Portal User Access for Contacts

(Customers/Vendors/Contacts/Partners)

Odoo App for Quick Create Portal User Access / Grant Access

This app allows your contact creation admin to quickly create portal access for contacts from form view and list view in mass as shown in below screenshots.

Main Features:

  • Allow Contact Manager to create quick portal users access from contact.
  • Allow Quick Grant Access for contact from form view.
  • Allow Quick Grant Access for contact and it's all contacts from form view.
  • Allow Quick Grant access for multiple contacts from the list for only selected contacts.
  • For more details, please check below screenshots and watch the video.

Edition Compatibility:
